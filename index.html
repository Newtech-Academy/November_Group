HTML:

HTML (HyperText Markup Language) (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afișate într-un browser (sau navigator).

CSS:

Cascading Style Sheets este un standard pentru formatarea elementelor unui document HTML

PHP:

Hypertext Preprocessor este folosit pe scară largă în dezvoltarea paginilor și aplicațiilor web.

MySQL:

Bazele de date relationale inteleg SQL (Structured Query Language), un limbaj relativ simplu, folosit pentru solicitarea datelor.
